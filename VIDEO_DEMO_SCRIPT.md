## 🎥 Recording Setup

### Tools Needed
1. **Screen Recorder:**
   - Mac: QuickTime, ScreenFlow, or Loom
   - Windows: OBS Studio, Camtasia
   - Online: Loom, Screencast-O-Matic

2. **Audio:**
   - Decent microphone (even laptop mic works)
   - Quiet room
   - Audio editing: Audacity (free)

3. **Video Editing:**
   - iMovie (Mac, free)
   - DaVinci Resolve (free)
   - Adobe Premiere Pro (paid)

### Recording Tips

**Before Recording:**
1. Clear browser cache/cookies
2. Use clean browser profile (no extensions)
3. Zoom to 100% or 125% for readability
4. Close unnecessary tabs
5. Hide bookmarks bar
6. Prepare demo data (sample portfolios, trades)
7. Open all pages you'll visit in tabs
8. Test microphone levels
9. Write out your script word-for-word

**During Recording:**
1. Record in 1080p (1920x1080)
2. Speak slowly and clearly
3. Pause between sections
4. Use smooth mouse movements
5. Hover to highlight important elements
6. If you mess up, pause and restart that section
7. Record extra B-roll (random navigation)

**After Recording:**
1. Add intro/outro graphics
2. Add background music (low volume)
3. Add text overlays for key features
4. Add zoom-ins for important details
5. Speed up boring parts (loading, etc.)
6. Add transitions between sections
7. Color correct if needed

## 💡 Pro Tips

1. **Practice First:** Do a full dry run before recording
2. **Script It:** Write out exactly what you'll say
3. **Show, Don't Tell:** Let the UI do the talking when possible
4. **Real Data:** Use realistic numbers, not all perfect results
5. **Pacing:** Don't rush - let viewers absorb what they see
6. **Call to Action:** End with clear next steps
7. **Accessibility:** Add captions/subtitles
8. **Thumbnail:** Create eye-catching thumbnail with text
9. **Music:** Use royalty-free music (YouTube Audio Library, Epidemic Sound)
10. **Length:** Shorter is better - aim for 6-8 minutes max
